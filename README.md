# 白泽工具箱

An Electron application with React and TypeScript

## Project Setup

### 安装

```bash
$ pnpm install
```

### 加载资源

百度网盘链接：https://pan.baidu.com/s/1r5NRUZOyqUkTIYDnp_lt5A?pwd=p7h7

阿里网盘链接：https://www.alipan.com/s/WRjhfS4T2p7

将解压后的linux、macos、win文件夹分别复制到项目的resources文件夹下

### 开发

```bash
$ pnpm dev
```

### 构建

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
